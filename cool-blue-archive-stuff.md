A list of Blue Archive related stuff I (as in UnderSet) found online.

_Note:_ Not an exhaustive list, likely missing _lots_ of stuff. Very biased. Feel free to submit feedback in [Issues](https://github.com/UnderSet/underset.github.io/issues) or [Discussions](https://github.com/UnderSet/underset.github.io/discussions).

- **Story scenario tools _(see below for MomoTalk)_**
  - [oryxel1](https://github.com/oryxel1)/[**Base**](https://github.com/oryxel1/Base) ![GitHub top language](https://img.shields.io/github/languages/top/oryxel1/Base?style=flat-square) ![GitHub last commit](https://img.shields.io/github/last-commit/oryxel1/Base?style=flat-square)
    <br>_A replicated engine for Blue Archive visual novel system!_
- **MomoTalk chat generators**
  - [UnderSet](https://github.com/UnderSet)/[**MomoHTMLEditor**](https://github.com/UnderSet/MomoHTMLEditor) ![GitHub top language](https://img.shields.io/github/languages/top/UnderSet/MomoHTMLEditor?style=flat-square) ![GitHub last commit](https://img.shields.io/github/last-commit/UnderSet/MomoHTMLEditor?style=flat-square)
    <br>_.NET based editor to create HTML MomoTalk content._
    - Based off of/using [this AO3 workskin](https://archiveofourown.org/works/64821349); taking different approaches from other generators in a few ways
  - [u1805](https://github.com/U1805)/[**momotalk**](https://github.com/U1805/momotalk) ![GitHub top language](https://img.shields.io/github/languages/top/U1805/momotalk?style=flat-square) ![GitHub last commit](https://img.shields.io/github/last-commit/U1805/momotalk?style=flat-square)
    <br>_Blue Archive MomoTalk Editor_
    - [Hosted here](https://u1805.github.io/momotalk/); more modern editor, but playable students only
  - [ClosureTalk](https://github.com/ClosureTalk)/[**closure-talk**](https://github.com/ClosureTalk/closure-talk) ![GitHub top language](https://img.shields.io/github/languages/top/ClosureTalk/closure-talk?style=flat-square) ![GitHub last commit](https://img.shields.io/github/last-commit/ClosureTalk/closure-talk?style=flat-square)
    <br>_Stylized talk generator_
    - [Hosted here](https://closuretalk.github.io/); older editor, out of date with current updates but has non-playable students and characters as well
- **Game data sites** _(note: not particularly endorsing any of them myself [^1], I just think they're cool)_
  - [**SchaleDB**](https://schaledb.com/student)
  - _There's probably more as well, let me know!_

[^1]: Though granted, this is mostly more because I don't particularly care about meta and such in games these days.
